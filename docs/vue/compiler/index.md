# 同构渲染

|                | SSR  | CSR    | 同构渲染|
|----------------|------|--------|--------|
| SEO            | 友好 | 不友好 | 友好|
| 白屏问题       | 无   | 有     | 无|
| 占用服务器资源 | 多   | 少     | 中|
| 用户体验       | 差   | 好     | 好|


同构渲染分为首次渲染（首次访问或刷新页面）以及非首次渲染

>同构渲染中的首次渲染与SSR渲染的工作流程是一致的。
